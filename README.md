# Smartphone-Usage-Addiction-Analysis

# Introduction:

This project analyzes smartphone usage data of 7,500 users to understand screen time patterns, user behavior, and addiction levels.

# Objective:

•	 Analyze screen time across age and gender

•  Identify addiction levels (None, Mild, Moderate, Severe)

•  Understand usage intensity (Low, Medium, High)

•  Study impact on sleep, stress, and daily activities

# Dataset:

•  7,500 user records

 Includes:
 
•  Age, Gender

•  Screen time (daily, social media, gaming)

•  Sleep hours, stress level

•  App usage & notifications

•  Addiction level

# Tools Used:

•  Excel

•  Power Query

•  Power BI / Pivot Tables

# Data Cleaning (Power Query) & Data Imputation:

•  Removed duplicates and handled missing values

•  Standardized text and categories

Created new columns:

•  Total Screen Time

•  Usage Level (Low / Medium / High)

•  Verified calculated fields for accuracy

# Key Analysis:

•  Average screen time ≈ 12.8 hours/day

•  Average sleep ≈ 6.7 hours/day

•  Usage data is evenly distributed across users

•  Majority fall into Medium to High usage

# Key Insights:

•  Screen time is high and consistent across age and gender

•  Most users fall under Medium (4K+) and High usage (2.7K+)

•  Moderate and Severe addiction levels dominate

•  Sleep duration is lower for heavy users

•  High screen time is linked to increased stress and reduced productivitystress.
